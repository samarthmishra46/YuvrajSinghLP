## ğŸ¨ Customization

### Colors
The primary color scheme can be modified in `src/styles.css`:

```css
@theme {
  --color-primary: #7C3AED;  /* Purple */
  --color-secondary: #1F2937; /* Dark Gray */
}
```

### Content
All content is located in the component files:
- `src/components/Hero.tsx` - Hero section content
- `src/components/About.tsx` - About section content
- `src/components/WorkWithMe.tsx` - Services and process content
- `src/components/Footer.tsx` - Footer links and social media

### Images
Replace placeholder emojis in components with actual images:
- Update the profile image placeholder in `Hero.tsx` and `About.tsx`
- Add brand logos in the brands section of `Hero.tsx`
- Add proof/screenshot images in `WorkWithMe.tsx`
